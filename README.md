# Sistema de Chamados – Salesforce (Admin)

Projeto prático desenvolvido na Salesforce Platform com foco em
administração, automação de processos e análise de dados,
simulando um cenário real de atendimento ao cliente.

## Descrição do Projeto
Este projeto tem como objetivo criar um sistema de chamados
utilizando recursos nativos do Salesforce, permitindo o
gerenciamento completo de solicitações de clientes,
desde a abertura até o encerramento do chamado.

## Funcionalidades
- Criação de objeto personalizado **Chamado**
- Configuração de campos personalizados (Status, Prioridade, Cliente, Descrição)
- Relacionamento do objeto Chamado com Conta/Cliente
- Personalização de Page Layout e Lightning Record Page
- Automação de atualização de status utilizando **Salesforce Flow**
- Criação de relatórios por status, prioridade e cliente
- Dashboard com indicadores de atendimento

## Tecnologias Utilizadas
- Salesforce Platform
- Salesforce Flow
- Relatórios e Dashboards
- Service Cloud (simulado)

## Objetivo
Aplicar boas práticas de administração Salesforce,
organização de dados e automação de processos,
simulando um ambiente real de suporte ao cliente.
